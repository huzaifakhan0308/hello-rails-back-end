## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Command line or terminal
- IDE such as VsCode.
- Rails needs to be installed on your system.

### Setup

Clone this repository to your desired folder:
Example command:

```sh
  git clone https://github.com/huzaifakhan0308/hello-rails-back-end.git
  cd hello-rails-back-end

```

### Install

Example command:

```sh
  cd hello-rails-back-end
  bundle install (install all dependencies)

```

### Usage


### API
to get random greetings from the API use the following link:
```sh
 http://127.0.0.1:3000/api/v1/greetings

```
### Frontend
to get random greetings from the frontend follow the follwing steps:
```sh
 git clone https://github.com/huzaifakhan0308/hello-react-front-end/.git
 cd hello-react-front-end
 npm install
 npm start
```
